# RepoLens

RepoLens turns a JavaScript or TypeScript repository into an interactive map of understanding.

It scans source files, parses imports and symbols, builds a dependency graph, and shows useful explanations for developers joining a codebase.

## Features

- Local workspace scanning
- Public GitHub repository scanning
- JavaScript and TypeScript source parsing
- File, folder, and dependency graph visualization
- Clickable file details
- Imports, exports, functions, and classes
- Incoming dependency view
- Folder contents view
- Ranked search results
- External dependency summary
- Generated onboarding path
- Basic file risk scoring
- API health check

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React Flow
- Lucide React
- TypeScript Compiler API
